# CCI_LMI_Project Overview
The CCI_LMI_Project repository is the official "Source of Truth" for testing and preparation within the CCI Project Based Learning Program. Its primary goal is to standardize how different IoT protocols are analyzed, documented, and tested.

# Core Structure: Protocol-Specific Modules
The repository is organized into four distinct "Knowledge Silos." Each folder contains the specific test procedures, setup notes, and technical preparation required for a particular wireless standard:
