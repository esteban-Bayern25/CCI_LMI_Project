# CCI_LMI_Project Overview
The CCI_LMI_Project repository serves as the official "Source of Truth" for the research, testing, and preparation phases of the CCI Project-Based Learning Program. Its primary mission is to standardize the methodology for analyzing, documenting, and stress-testing diverse IoT protocols. 

This repository is designated as Appendix A in the project's final technical report and serves as the central hub for all raw data and procedural evidence.

# Core Structure: Protocol-Specific Modules
The repository is organized into four distinct modules, each containing the specific test procedures, hardware setup notes, and technical preparation required for a particular wireless standard:
